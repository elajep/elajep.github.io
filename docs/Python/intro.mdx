# Introduzione a Python

Python è un **linguaggio di programmazione** ad alto livello, noto per la sua **leggibilità** e semplicità. È molto usato per lo sviluppo web, data science, intelligenza artificiale e automazione.

## Caratteristiche principali

- Sintassi chiara e semplice
- Tipizzazione dinamica
- Supporto a diversi paradigmi: procedurale, orientato agli oggetti e funzionale
- Ampia libreria standard e ecosistema di pacchetti

:::note
Python è ottimo per chi inizia a programmare grazie alla sua sintassi leggibile e alla comunità attiva.
:::

## Primo esempio

Il classico “Hello, World!” in Python:

```python
print("Hello, World!")
```

## Variabili e tipi

In Python non serve dichiarare il tipo delle variabili, basta assegnarle:

```python
nome = "Alessandro"   # stringa
eta = 25              # intero
pi = 3.14             # float
```

Puoi usare il codice inline per riferirti a funzioni o variabili: ad esempio, `print(nome)` stampa il contenuto della variabile nome.

## Condizioni

```python
if eta >= 18:
    print("Sei maggiorenne")
else:
    print("Sei minorenne")
```

## Cicli

```python
for i in range(5):
    print(i)
```

:::note
Ricorda: in Python l’indentazione è fondamentale. Spazi o tab al posto giusto definiscono i blocchi di codice.
:::
